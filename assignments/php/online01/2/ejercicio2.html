Tutorial XAMPP y PHP

Instalando el servidor XAMPP

¿Qué es XAMPP?

Más que un servidor, XAMPP es un paquete de software que incluye lo mínimo necesario para servir una web:

  • el servidor Apache,
  • un sistema de gestión de base de datos (MariaDB a partir de la versión 5.6.15; MySQL en versiones anteriores) y
  • soporte para dos lenguajes de servidor: php y perl.

¿Dónde conseguirlo?

Se puede descargar desde la misma web oficial, apachefriends, donde hay disponibles versiones para Windows y Linux.

Desde la página principal, navegamos hasta la opción Descargar en el menú de pestañas:

Nos lleva a SourceForge, donde podemos elegir qué versión descargar en función de nuestro sistema operativo:

Requisitos

En principio no es necesario instalar ningún paquete de software adicional; todo lo necesario está incluido en la descarga oficial.

Windows

Para instalar la versión de Windows sólo hay que seguir los siguientes pasos:

  1. Escoge la opción correspondiente
  2. Navega hasta la carpeta donde hayas descargado el ejecutable.
  3. Haz doble click sobre el archivo y sigue las instrucciones de instalación.
  4. se inicia en modo gráfico
  5. Los archivos de nuestra(s) página(s) se guardan en C:\xampp\htdocs

Linux

En Linux podemos instalar XAMPP de dos formas, en modo gráfico, mediante el explorador de archivos de nuestra distribución y el ratón, o por consola, ejecutando una serie de comandos. Esta última opción es la única posible si estamos trabajando en un servidor sin interfaz gráfica, por ejemplo.

  1. Si quieres instalarlo en modo gráfico,
    1. Navega hasta la carpeta donde hayas descargado el ejecutable.
    2. haz click derecho sobre el archivo y elige la opción abrir con el instalador de paquetes.
  2. Si quieres instalarlo en modo texto:
    1. abre una terminal de comandos
    2. navega hasta el directorio donde hayas guardado la descarga cd directorio/
    3. ejecuta con permisos de superusuario dpkg -i xampp-installer.run
    4. sigue las instrucciones en pantalla
  3. Los archivos de nuestra(s) página(s) se guardarán en el directorio/opt/lampp/htdocs
  4. /opt/lampp se inicia en modo gráfico desde el menú, o desde terminal con sudo /opt/lampp/xampp start

PHP

PHP es un lenguaje utilizado para crear páginas web con contenido dinámico. Entre otras cosas, permite:

  • Recibir y procesar datos de formularios
  • Comunicarse y trabajajr con bases de datos
  • Utilizar y gestionar sesiones de usuario
  • Encriptar los datos y manejarlos con seguridad

Ejecutar código PHP

PHP es un lenguaje de servidor; eso quiere decir que se ejecuta en el lado del servidor (el programa que sirve la página o aplicación web) y no es visible en el lado del cliente. Abrir directamente en el navegador un archivo escrito en php sólo mostrará las etiquetas html que contenga, pero nada codificado en php.

¿Cómo podemos ver el resultado de un código PHP? Alojándolo en un servidor, que se encargará de procesarlo y enviar al cliente los datos resultantes.
