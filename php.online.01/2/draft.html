Instalando el servidor XAMPP

¿Qué es XAMPP?

Más que un serviddor, XAMPP es un paquete de software que incluye lo mínimo necesario para servir una web:

  • el servidor Apache,
  • un sistema de gestión de base de datos (MariaDB a partir de la versión 5.6.15) y
  • soporte para dos lenguajes de scripting: php y perl.

¿Dónde conseguirlo?

Desde la misma web oficial, apachefriends, donde hay disponibles versiones para Windows y Linux.

Requisitos

En principio no es necesario instalar ningún paquete de software adicional; todo lo necesario está incluido en la descarga oficial.

Windows

  1. Escoge la opción correspondiente
  2. Navega hasta la carpeta donde hayas descargado el ejecutable.
  3. Haz doble click sobre el archivo y sigue las instrucciones de instalación.
  4. se inicia en modo gráfico
  5. Los archivos de nuestra página se guardan en C:\xampp\htdocs

Linux

  1. Escoge la opción correspondiente
  2. Navega hasta la carpeta donde hayas descargado el ejecutable.
  3. Si quieres instalarlo en modo gráfico, haz click derecho sobre el archivo y elige la opción abrir con el instalador de paquetes.
  4. Si quieres instalaro en modo texto, mediante la terminal:
    1. abre una consola
    2. navega hasta el directorio cd directorio/
    3. ejecuta con permisos de superusuario dpkg -i xampp-installer.run
    4. sigue las instrucciones en pantalla
  5. Los archivos de nuestra página se guardan en el directorio es /opt/lampp/htdocs
  6. /opt/lampp se inicia en modo gráfico desde el menú, o desde terminarl con sudo /opt/lampp/xampp start
